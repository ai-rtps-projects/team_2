🛍️ AI Shopping Assistant Chatbot

An intelligent E-commerce Chatbot built using Python (Flask) that helps users discover, compare, and choose products through a conversational interface.

---

📌 Project Overview

In modern e-commerce platforms, users often struggle to find the right product among thousands of options. This chatbot simplifies the shopping experience by allowing users to interact using natural language queries.

💬 Example queries:

- "Find headphones under $50"
- "Compare laptops"
- "Show top-rated products"

---

🚀 Features

- 🔍 Product Search using keywords
- 💰 Budget-based filtering
- ⭐ Rating-based recommendations
- 📊 Product comparison
- 🤖 Conversational interface
- 🌐 Web-based UI

---

🛠️ Tech Stack

- Backend: Python, Flask
- Frontend: HTML, CSS, JavaScript
- Data Storage: JSON
- Concepts: NLP (rule-based), intent detection

Install Dependencies

Run the Project

Open in Browser

http://127.0.0.1:5000
